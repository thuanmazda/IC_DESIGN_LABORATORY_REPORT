# EE3165 - IC Design: Lab 2 - Synthesis and Standard Cell

This repository contains the RTL designs, synthesis scripts, and gate-level simulation testbenches for **Lab 2: Synthesis and Standard Cell** as part of the IC Design course at Ho Chi Minh City University of Technology (HCMUT).

UPDATE: 10:40AM 21-04-2026

## Project Overview
This project covers the complete digital design flow from RTL design to logic synthesis and gate-level simulation (GLS). Using Cadence tools, the lab investigates the characteristics and performance of standard cells in a 45nm technology library across eight different Process, Voltage, and Temperature (PVT) corners.

### Experiments Included:
1. **Combinational Cells Characterization**: Synthesizing basic logic gates (NOT, OR2, NAND2, NOR2, MUX, and FULL ADDER). The experiment extracts and evaluates critical metrics including **power**, **area utilization**, and **propagation delay** across 8 distinct process corners (varying between Fast/Slow process, 1.0V/1.2V VDD, and LVT/HVT thresholds).
2. **Sequential Cell Characterization**: Synthesizing a D-Flip Flop (D-FF) across the 8 corners. Analyzing timing parameters including clock-to-Q delay and rigorously verifying sequential constraints such as **setup time** and **hold time** using SDF back-annotation.

## Repository Structure
* `00_src/`     : SystemVerilog RTL design files.
* `01_tb/`      : SystemVerilog Testbench files configured for Gate-Level Simulation with SDF annotation.
* `02_reports/` : Synthesis reports (area, power, timing, and QoR) generated by Genus.
* `03_outputs/` : Synthesized gate-level netlists (`.sv`) and Standard Delay Format (`.sdf`) files.
* `scripts/`    : TCL scripts used for automating the synthesis process in Cadence Genus.
* `docs/`       : Detailed lab report and waveform simulation screenshots verifying delays and timing checks.

## Tools & Technology
* **HDL:** SystemVerilog
* **Synthesis Tool:** Cadence Genus
* **Simulation Tool:** Cadence Xcelium / SimVision
* **Scripting:** TCL (Tool Command Language)
* **Technology Node:** 45nm Standard Cell Library

## Verification
Functional correctness and timing behaviors of all synthesized netlists have been verified using Gate-Level Simulation (GLS). The testbenches utilize the `$sdf_annotate` system task to back-annotate delay information generated by Genus. Waveform analysis is used to measure absolute propagation delays and confirm that critical timing constraints (setup/hold) are met under specific PVT corner conditions.
